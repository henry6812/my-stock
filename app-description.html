My Stock Portfolio App

文件日期：2026-03-03

這是一個以 local-first 為核心的個人資產管理 Web App，主要用於追蹤台股/美股持股現值， 同時整合現金帳戶、收支、預算與趨勢分析，支援 PWA 安裝及跨裝置同步。

產品定位

面向個人與家庭財務管理情境，提供「持股 + 現金 + 支出 + 預算」的一站式檢視， 讓使用者可以在手機或桌機快速掌握總資產與金流變化。

PWA 離線優先 跨裝置同步

核心功能

  • 持股管理：新增/編輯/刪除持股，支援排序、標籤、持有人欄位。
  • 即時估值：整合美股/台股價格來源與 USD/TWD 匯率，計算總現值與報酬。
  • 價格更新策略：台股採多層 fallback（Finnhub → TWSE → TPEX 快照/官方 API/代理）。
  • 趨勢分析：提供 24h / 1w / 1m 等維度的現值走勢圖。
  • 現金帳戶：維護帳戶餘額與帳戶持有人，納入總資產統計。
  • 收支與預算：支援支出分類、定期支出、預算設定與分析儀表。
  • 雲端同步：Google 登入 + Firebase Firestore，保留 local-first 使用體驗。

系統架構

前端層
  • React 19 + Vite
  • Ant Design UI
  • Recharts 圖表
  • animejs 動畫
資料層
  • Dexie / IndexedDB
  • Firestore 雲端同步
  • 價格/匯率 Provider 抽象
  • 本地快照 fallback

資料模型（IndexedDB）

  • holdings：持股資料
  • price_snapshots：價格快照（用於趨勢）
  • fx_rates：匯率資料
  • sync_meta：同步狀態
  • expense_entries：支出資料（可擴充）

部署與運維重點

  • 已具備 GitHub Pages 佈署流程（CI workflow）。
  • 需要設定 Finnhub API Key 與 Firebase 相關環境變數。
  • 建議啟用 Firebase App Check（reCAPTCHA v3）降低濫用風險。

總結

My Stock Portfolio App 具備完整的個人資產追蹤核心能力，並透過 local-first + 雲端同步策略同時兼顧可用性與資料一致性。 以目前架構可持續擴充至更多市場、更多資產類型與更進階的財務分析需求。
