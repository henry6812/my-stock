My Stock Portfolio App

文件主題：核心賣點說明 | 日期：2026-03-03
一句話定位： 這是一款「離線可用 + 跨裝置同步」的個人資產管理 App，將股票、現金與收支整合在同一個畫面，幫助使用者快速掌握真實總資產。

核心賣點

  • Local-first 體驗：資料優先儲存在 IndexedDB，本機可立即操作，網路不穩時仍可查看與管理持股。
  • 跨裝置同步：透過 Google 登入與 Firebase Firestore，同一份資料可在手機與桌機之間持續同步。
  • 多來源報價容錯：台股採用 Finnhub、TWSE、TPEX 快照與 API 的多層 fallback，降低單一來源失效風險。
  • 資產全貌整合：同時管理持股、現金帳戶、收支與預算，不只看投資，也看整體財務健康。
  • 可安裝式 PWA：支援桌面與手機安裝，打開速度快，使用流程接近原生 App。

為什麼這些賣點有價值

穩定性

即使 API 波動或網路異常，仍可用本地資料與 fallback 機制維持主要功能。

效率

從持股更新到資產總覽都在同一個流程，減少切換工具與手動彙整成本。

可信度

本地快照、匯率資料與走勢記錄可回溯，便於檢視資產變化與決策依據。

可擴充性

現有架構已具備資料抽象層，未來可擴展更多市場與資產類型。

技術支撐亮點

React + Vite Dexie / IndexedDB Firebase Firestore PWA Recharts

前端效能、離線能力、雲端同步與視覺化分析完整結合，形成可持續演進的產品基礎。

結論

My Stock Portfolio App 的核心競爭力在於：離線優先的穩定使用體驗、 多來源資料容錯與完整資產視圖。它不是單純看股價的工具，而是可落地使用的個人財務中樞。
